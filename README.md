##HomeBuilder 2.0##


**WHAT IS HomeBuilder?**

HomeBUilder is a MFC-based house management/storage system. 

**HOW TO USE IT?**

Open the program and input the following information for each teacher:

- Name, Gender, Title (Assistant, Instructor, Associated Professor, Professor)

- Date of Birth, Date of Working, Highest Degree;

- Family info: families' names, Date of Birth, etc;

HomeBuilder 2.0 has implemented all basic functions, like GUI, data collection and display functionalities. One possible improvement could be the database function, i.e, feeding the data to DBMS, mySQL, SQLite3, etc.
